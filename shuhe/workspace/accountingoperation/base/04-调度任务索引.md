## 调度任务框架

### SchedulerPlus 框架

任务使用 **SchedulerPlus** 框架进行调度管理（基于 SchedulerX）。

### 任务注解

使用 `@JobInfo` 注解标记任务类：

```java
@JobInfo(jobName = "taskName")
public class TaskJob extends JobExecutor {
    @Override
    public JobResult execute(String externalData) {
        // 任务逻辑
        return new JobResult(JobCodeMessages.SUCCESS_CODE, JobCodeMessages.SUCCESS_MESSAGE);
    }
}
```

### 任务继承模式

| 模式 | 基类 | 说明 |
|-----|------|------|
| 简单任务 | `JobExecutor` | 实现 `execute(String externalData)` 方法 |
| 分片任务 | `BaseJob<T>` | 实现 `queryNeedHandleData()`, `distributedLockKey()`, `process()` 方法 |

### 分片任务接口

```java
public abstract class BaseJob<T> extends JobExecutor {
    // 查询待处理数据
    protected abstract List<T> queryNeedHandleData(Integer shardingTotal, Integer shardingItem, int limit);

    // 构建分布式锁 Key
    protected abstract String distributedLockKey(T bo);

    // 处理单条数据
    protected abstract void process(T bo);
}
```
