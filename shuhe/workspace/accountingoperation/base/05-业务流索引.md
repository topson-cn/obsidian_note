## 业务流说明

### BizFlow 流程引擎

**框架:** BizFlow 业务流程引擎

**核心概念:**
- **BizKey** - 业务流程唯一标识
- **流程节点** - 流程中的处理步骤
- **流程分支** - 根据条件选择不同流程路径
- **循环结构** - 支持卡循环、订单循环等

**节点类型:**
- 前置检查节点 - 验证前置条件
- 业务处理节点 - 执行业务逻辑
- 分支判断节点 - 流程分支选择
- 等待节点 - 等待异步结果
- 结果处理节点 - 处理执行结果
- 通知节点 - 发送通知

## 流程查询

### 通过 MCP 查询业务流

使用 `mcp__sh_biz_plan__plan_detail_query_by_bizkey` 工具查询业务流详情：

```java
// 查询人工扣款提交流程
mcp__sh_biz_plan__plan_detail_query_by_bizkey(
    bizKey: "manualDeductSubmitFlow",
    version: "latest"
)
```

### 节点实现类匹配

BizFlow 节点通过 Spring Bean 名称匹配到实现类：

```java
@Service("manualDeductSubmitDeductProcess")
public class ManualDeductSubmitDeductProcess extends BaseMyBizFlowProcess<ManualDeductContext> {
    // 节点实现
}
```

**匹配规则:**
1. Spring Bean 名称 = 节点编码
2. 继承 `BaseMyBizFlowProcess` 或 `BaseBizFlowProcess`
3. 实现 `process()` 方法
