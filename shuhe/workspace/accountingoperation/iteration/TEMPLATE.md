## 📁 目录结构

```

│   ├── 202502-法诉自动入账-设计文档.md
```

## 📌 使用说明

1. **创建新功能文档目录**
   ```bash
   mkdir -p "iteration/{年}{月}-{功能名}"
   ```

2. **复制模板文件**
   ```bash
   cp TEMPLATE.md "iteration/{年}{月}-{功能名}/{年}{月}-{功能名}-文档类型.md"
   ```

3. **填写文档内容**
   - 根据模板结构填写具体内容
   - 删除不需要的章节
   - 添加特定功能的内容

4. **引用基线文档**
   - 使用相对路径引用基线文档
   - 例如: `[项目工程结构](../base/01-项目工程结构.md)`
