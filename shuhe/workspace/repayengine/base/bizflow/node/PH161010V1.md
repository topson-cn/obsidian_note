## 功能说明

将还款处理任务提交到线程池，异步启动重资产还款处理主流程。同步流程到此节点后不再等待后续处理结果。

### 核心职责
1. **异步任务构建**: 构建RepayNewBizFlowTask
2. **任务提交**: 提交到还款申请处理线程池
3. **失败处理**: 提交失败返回PAUSED（支持重试）

## 处理流程

```mermaid
flowchart TD
    A[开始] --> B[构建RepayNewBizFlowTask]
    B --> B1["bizKey=HEAVY_V4_0_1_HANDLE"]
    B1 --> B2[设置bizSerial/context/triggerService]
    B2 --> C[repayApplyProcessExecutor.execute]
    C --> D{提交成功?}
    D -->|成功| E[返回SUCCESS]
    D -->|失败| F[记录错误日志]
    F --> G[返回PAUSED]
```

## 核心业务逻辑

### 1. 异步任务构建
- bizKey: `BIZFLOW_BIZ_KEY_HEAVY_V4_0_1_HANDLE`
- bizSerial: 从repayContext获取
- 非阻塞，主流程不等待执行结果

### 2. 失败处理
- 返回 PAUSED（非FAILED），允许重试

## 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| CjjServerException | 返回PAUSED |
| 线程池满 | 返回PAUSED |
