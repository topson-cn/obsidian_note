# PS150001 - 还款结果查询

| 属性 | 值 |
|------|-----|
| **处理器代码** | PS150001 |
| **节点名称** | 还款结果查询 |
| **节点类型** | PROCESS |
| **所属流程** | [[出让订单还款流程V1]] |
| **执行阶段** | 结果处理阶段 |

## 功能说明

查询还款提交的执行结果，确认还款是否成功。

### 核心职责
1. **结果查询**: 查询还款执行结果
2. **状态更新**: 根据结果更新还款状态

## 异常处理

| 异常场景 | 错误类型 | 处理方式 | 影响 |
|----------|----------|----------|------|
| 查询超时 | RuntimeException | 重试1次，间隔60秒 | 失败后暂停 |
| 结果异常 | RuntimeException | 重试1次，间隔60秒 | 失败后暂停 |

**节点异常策略**:

- **重试间隔**: 60秒
- **失败后状态**: PAUSED (暂停)
