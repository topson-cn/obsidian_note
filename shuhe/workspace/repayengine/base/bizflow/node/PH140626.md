## 功能说明

组装还款模式策略决策引擎的输入参数，构建Facts Map用于策略决策。

### 核心职责
1. **参数组装**: 从还款上下文提取策略所需的各维度数据
2. **计划列表构建**: StagePlanItem → RepayStrategyInputPlanBo

## Facts Map 构建

| Fact Key | 来源 | 说明 |
|----------|------|------|
| UID | repayApplyContext | 用户ID |
| ASSET_BANK | currentRepaymentBill | 资方银行枚举 |
| ASSET_ID | currentRepaymentBill | 资方ID |
| STRATEGY_PARAM_REPAY_WAY | repayApplyBo | 还款方式 |
| STRATEGY_PARAM_PAY_TYPE | 第一个payTool | 支付类型 |
| DEDUCT_CHANNEL_DEDUCT_WAY | fundConfig查询 | 扣款方式 |
| DEDUCT_CHANNEL_ORDER_NO | stageOrderItem | 订单号 |
| STRATEGY_PARAM_PLAN_LIST | stagePlanItems转换 | 计划列表 |

### 计划列表转换 (buildPlanInfoList)
- planNo ← stagePlanNo
- repayScene ← repayScene
- fundPayOff ← fundPayOff（来自PH140624）

## 异常处理

| 异常场景 | 错误码 |
|----------|--------|
| 订单列表为空/多个订单 | REPAY_LIGHT_ASSET_ORDER_ERROR |
